   Design comes from the understanding of life, using simple way to design high cohesion products, SuperBuilder is an open source rapid development platform, based on the design idea of RABC2.0 design, 100% open source code. The front end is improved for the DWZ front-end open source framework, and the code security standard is designed, which is based on the spring framework. The data structure is designed and implemented through Alibaba Druid framework.

2, springframework3.2.0.Release
3. Test framework junit4.11
